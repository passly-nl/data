# `@passly-nl/data`

Full API implementation for Passly.nl.

## ⭐️ Prerequisites

- Bun >= 1.3.0
- Node >= 25

## 🚀 Getting started

1. Run `bun install` in the root of the project.
2. Run `bun run build` to build the project.
